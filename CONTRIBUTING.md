# Guía de Contribución

¡Gracias por ayudar a Santa! Para mantener la integridad del sistema:

1. Asegúrate de que todos los tests científicos pasen: `npm run test:scientific`.
2. No modifiques la precisión decimal de la telemetría sin aprobación previa.
3. Si encuentras un fallo en el sistema de auditoría, repórtalo inmediatamente vía Sentry.
